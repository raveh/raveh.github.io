树莓派

去年买的树莓派，8GBi版本，安装了pi64的操作系统，没有找到好的64位系统的镜像网站，更新系统和安装软件的速度特别慢，想来想去不知道干什么用，今天把树莓派利用起来了，使用了frp联通了网络,lighttpd,hugo生成静态网页,目前看起来不错。

安装软件：

sudo apt install vim git hugo lighttpd

配置防火墙策略：

$ ufw status
Status: active

To                         Action      From
--                         ------      ----
22                         ALLOW       192.168.3.0/24
80                         ALLOW       Anywhere
443                        ALLOW       Anywhere
80 (v6)                    ALLOW       Anywhere (v6)
443 (v6)                   ALLOW       Anywhere (v6)

配置frp客户端：

[common]
server_addr = xxx.xxx.xxx.xxx
server_port = xxxx

[web_pi_raveh_net]
type = http
local_port = 80
custom_domains = pi.raveh.net
host_header_rewrite = pi.raveh.net
